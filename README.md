# SimplyLoopy
## About
Subdivision surface modelling is a fun and efficient way to create objects with nice topology, but adds a lot of unnecessary edge-loops, generating high-poly meshes. This Blender addon simplifies a mesh using its edge-loops, while keeping important details intact!
